# 05 语法暗黑森林：6个语法刺客

对应文章：《C语言语法暗黑森林：6个让单片机原地爆炸的语法刺客》

## 文件
- `syntax_traps.c` —— 6 个语法陷阱及安全写法。

## 构建与运行
```bash
make run     # Makefile 已加 -Wempty-body
```

## 覆盖的陷阱
1. 复杂函数声明 → 用 `typedef`
2. 运算符优先级（`~`、`<<`、位运算加括号）
3. `if` 后误加分号
4. `switch` 缺 `break` 的 fall-through
5. 可变参数/格式串类型（显式强转）
6. 悬空 `else`（大括号消除歧义）
