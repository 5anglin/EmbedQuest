# 08 库函数陷阱：库函数调用生存指南

对应文章：《C语言库函数陷阱：库函数调用生存指南》

## 文件
- `library_traps.c` —— 4 类库函数陷阱的安全用法（运行时会在当前目录生成临时文件并自动清理）。

## 构建与运行
```bash
make run
```

## 覆盖的陷阱
1. `getchar()` 返回 `int`，不能用 `char` 承接
2. 顺序文件更新：`rb+` + `fseek` + 先读后写 + `fflush`
3. 缓冲区：`setvbuf` 定制，缓冲区生命周期长于文件流
4. `errno`：调用后立即保存，用 `strerror` 报告
