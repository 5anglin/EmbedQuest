# 11 万物皆可状态机（FSM）

对应文章：《万物皆可状态机：解密状态机 (FSM)，轻松驾驭复杂系统》

## 文件
- `fsm_switch.c` —— 经典 `switch-case` 状态机。
- `fsm_table.c` —— 函数指针表（状态表）状态机。
- `led_fsm.c` —— 实战：用 FSM 控制 LED（灭→亮→闪烁→灭 循环），含硬件抽象层。

## 构建与运行
```bash
make run
```

## 要点
- 状态少时用 `switch-case`；状态/事件多时用状态表更清晰、易扩展。
- 进入/退出/转换动作分离；中断里只设事件，主循环/任务里跑状态机。
