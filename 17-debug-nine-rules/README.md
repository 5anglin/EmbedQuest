# 17 Debug 九大法则

对应文章：《Debug：玄学还是科学？九大法则助你快速定位软件缺陷》

## 文件
- `debug_helpers.h` —— 调试辅助宏：`TRACE` / `DBG_ASSERT` / `CHECKPOINT`（体现"观察而非空想""检查假设"）。
- `bisect_demo.c` —— **可运行**：用二分查找"分而治之"定位数组中第一个越界值。

## 构建与运行
```bash
make run
```

## 九大法则速记
1. 读懂系统（手册为友） 2. 重现Bug 3. 眼见为实（停止空想）
4. 分而治之 5. 每次一改 6. 记录日志（保留审计轨迹）
7. 检查假设 8. 获取新视角（求助/换人看） 9. 修好确认（并防回归）
